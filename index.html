SMARTSTUDY lottery system 2015-12-29

We used this adhoc-system to draw winners for end-of-year-2015 party at SMARTSTUDY.

> > > L A U N C H ! < < <

Keyboard Shortcuts

  • Drag : Rotate view
  • Zoom : Zoom view
  • R : Run
  • S : Stop
  • 1 : 0.25x speed
  • 2 : 1x speed
  • 3 : 4x speed
  • 4 : 16x speed
  • 5 : 100x speed

You can also pick a object and throw it. lol
